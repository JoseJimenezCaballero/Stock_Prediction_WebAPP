About

This app was built using a combination of programming languages and libraries. It uses a linear regression model to predict if the next return of the given interval will be positive or negative. The model is dynamically trained every time it is called which ensures the most recent data is used. It uses a training model of 80% of the inputted data and runs tests on 20% of the data. The data the model uses is directly downloaded from yahoo finance using their api.
